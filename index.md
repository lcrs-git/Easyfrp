
#### 快速开始
你只需要修改frpc.ini 中的服务器地址以及端口等，例如服务器地址 s1.frp.asia 端口7004 

```
[common] 
server_addr = s1.frp.asia
server_port = 7004
token = loz7wS
```
修改好配置文件，运行客户端就可以使用了！推荐使用stcp方式连接！
