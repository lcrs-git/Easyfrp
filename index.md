
#### 快速开始
要使用frp，你只需要修改frpc.ini 中的服务器地址以及端口，例如服务器地址 s1.frp.asia 端口7004

```
[common] 
server_addr = s1.frp.asia
server_port = 7004
token = loz7wS
```
连通后，客户端即可使用stcp方式连接了，快来薅羊毛！
