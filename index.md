
# 快速开始
### 修改frpc.ini

`[common] `

`server_addr = 服务器地址`

`server_port = 端口`

`token = 密码`

`[ssh] # 示例`

`type = tcp`

`local_ip = 127.0.0.1#你的本地服务地址`

`local_port = 22#本地端口`

`remote_port = 6000#远程端口`
