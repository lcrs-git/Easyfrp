
#### 快速开始
修改frpc.ini 中服务器地址以及端口，例如服务器地址 s1.frp.asia 端口7004

```
[common] 
server_addr = s1.frp.asia
server_port = 7004
token = 密码
```
获取token之后，客户端即可使用stcp方式连接
